# Подсказка по гиту

## Инициализация репозитория

```sh
git init
```

## Статус репозитория

```sh
git status
```
# Добавить к индексации
```sh
git add
```
# Зафиксировать изменения
```sh
git commit -m
"messege"
```
# Посмотреть лог в полном виде 
```sh
git log
```
# Посмотреть лог в сокращенном виде
```sh
git log --oneline
```
# Перемещение по веткам
```sh
git checkout
```
# Отоброжение всех веток 
```sh
git branch
```
